# Fuck YouTube Premium for Orion

Firefox WebExtension for **Orion on iOS**.

## Install

1. AirDrop `youtube-mobile-orion-firefox-2.0.3.zip` (or `.xpi`).
2. Orion → **••• → Settings** → enable **Firefox extensions**.
3. **••• → Extensions → + → Install from file** (replace older version).
4. Open YouTube; allow the extension if prompted.

## Behavior (2.0.3)

- **Single-tap** hamburger opens the guide (no more double-tap fight)
- **Shorts** removed from the guide drawer and feeds; `/shorts` redirects home
- No floating pill / mini-guide rail — burger navigation only
- Miniplayer dismissed; play stays inline; PiP only via status-light tap
- Light edge padding
